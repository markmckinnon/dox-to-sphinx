iOS Analyzer (iLEAPP)

Table of Contents

  • Overview
  • Using the Module
  • Seeing Results

Overview

The iOS Analyzer ingest module runs iLEAPP (https://github.com/abrignoni/iLEAPP) and converts the results into results that can be viewed in Autopsy.

Using the Module

Select the checkbox in the Ingest Modules settings screen to enable the IOS Analzyer (iLEAPP) module. The module will run on .tar/.zip files found in a logical files data source or a disk image.

Seeing Results

Results from the iOS Analyzer module will appear in the Tree Viewer under Results->Extracted Content.
