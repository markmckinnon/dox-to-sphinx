Autopsy Developer's Guide

Overview

This is the Developer's Guide for the open source Autopsy platform. Autopsy allows you to examine a hard drive or mobile device and recover evidence from it. This guide should help you with using Autopsy. There is a seperate developer's guide for writing Autopsy modules.
